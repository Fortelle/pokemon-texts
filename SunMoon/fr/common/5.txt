Tournoi Amical
Bienvenue au Coin Combats ! Ici, vous pouvez vous livrer à des Combats Link avec les joueurs du monde entier. Vous pouvez également vous inscrire sur le Pokémon Global Link pour participer à diverses Compétitions !
En Combat Libre, chaque Pokémon de votre Équipe Combat doit être d'une espèce différente et tenir un objet différent. Vous pouvez choisir d'utiliser une équipe avec des Pokémon spéciaux ou sans Pokémon spéciaux.
Lors d'un combat en mode Rang, vous ne pouvez pas utiliser de Pokémon spéciaux. Chaque Pokémon de votre Équipe Combat doit être d'une espèce différente et tenir un objet différent. Le règlement des Combats Spéciaux et des Combats WCS change selon la saison.
Pour prendre part aux Compétitions Internet, vous devez vous inscrire sur le site Internet du Pokémon Global Link. Pour en savoir plus sur les Compétitions et leur règlement spécifique, vous pouvez visiter le site du Pokémon Global Link : http://www.pokemon-gl.com/
Pour prendre part aux Tournois Amicaux, vous devez vous inscrire sur le site Internet du Pokémon Global Link. Pour en savoir plus sur les Tournois Amicaux et leur règlement spécifique, vous pouvez visiter le site du Pokémon Global Link : http://www.pokemon-gl.com/
Pour plus d'informations sur ce règlement, vous pouvez visiter le site Internet du Pokémon Global Link : http://www.pokemon-gl.com/ Vous pouvez également scanner le QR Code pour vous rendre directement sur le site.
Vous pouvez choisir d'envoyer automatiquement les vidéos enregistrées après chaque combat en mode Rang vers le serveur, pour que leur code puisse être potentiellement publié sur le site du Pokémon Global Link. Il est aussi possible de participer aux Combats en mode Rang en refusant l'envoi de Vidéos de Combat.
Les Pokémon participant sont inscrits avant le premier combat.
