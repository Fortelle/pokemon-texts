Les Boîtes sont des réceptacles où vous pouvez déposer vos Pokémon. Une fois déposés, vous pouvez les organiser comme bon vous semble. Il est possible de vous connecter à vos Boîtes depuis le PC de n'importe quel Centre Pokémon.
Il y a trois modes de sélection dans les Boîtes. Appuyez sur le bouton X pour passer d'un mode à l'autre. Le premier mode permet de ranger les Boîtes, par exemple en intervertissant les Pokémon de la Boîte avec ceux de votre équipe, ou en les réorganisant selon votre goût.
Le deuxième mode permet de ranger les objets tenus par les Pokémon. Vous pouvez ainsi donner un objet à chaque Pokémon, ou intervertir les objets tenus par ces Pokémon.
Le troisième mode permet de créer jusqu'à six Équipes Combat, composées de Pokémon prêts à se battre. Il est possible d'enregistrer le même Pokémon dans plusieurs Équipes Combat, ce qui vous permet d'essayer toutes sortes de variations.
START et SELECT permettent de changer les contrôles. Les contrôles par défaut sont utilisés quand le curseur est rouge. Le curseur bleu permet d'intervertir rapidement les Pokémon. Enfin, le curseur vert permet de sélectionner plusieurs Pokémon d'un seul coup.
Le bouton Y permet de chercher des Pokémon selon les critères de votre choix, par exemple en fonction de leur nom, de leur type ou des capacités qu'ils connaissent.
