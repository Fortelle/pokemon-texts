Il s'agit d'un système permettant de synchroniser Pokémon Rubis Oméga ou Pokémon Saphir Alpha avec le site Internet du Pokémon Global Link : http://www.pokemon-gl.com/
Vous pouvez utiliser la Synchro-Jeu pour partager votre sauvegarde avec le Pokémon Global Link.
Sur le Pokémon Global Link, vous pouvez consulter l'état de votre sauvegarde de Pokémon Rubis Oméga ou de Pokémon Saphir Alpha, ou bien jouer à des jeux qui vous feront gagner des objets. Ces objets pourront ensuite être envoyés dans Pokémon Rubis Oméga ou Pokémon Saphir Alpha.
Pour utiliser la Synchro-Jeu, vous devez : 1) Sélectionner « Générer votre ID de Synchro-Jeu » dans le menu de la Synchro-Jeu.
2) Vous connecter sur le site Internet du Pokémon Global Link http://www.pokemon-gl.com/ et y enregistrer cet ID de Synchro-Jeu. Une fois ceci effectué, la Synchro-Jeu est prête à être utilisée !
Veuillez noter que pour enregistrer votre ID, vous aurez besoin d'un compte sur le site Internet du Pokémon Global Link, exploité par The Pokémon Company et ses sociétés affiliées. Pour plus d'informations, veuillez consulter le site Internet du Pokémon Global Link.
Il s'agit d'un système permettant de synchroniser Pokémon Rubis Oméga ou Pokémon Saphir Alpha avec le site Internet du Pokémon Global Link : http://www.pokemon-gl.com/
Vous pouvez utiliser la Synchro-Jeu pour partager votre sauvegarde avec le Pokémon Global Link.
Sur le Pokémon Global Link, vous pouvez consulter l'état de votre sauvegarde de Pokémon Rubis Oméga ou de Pokémon Saphir Alpha, ou bien jouer à des jeux qui vous feront gagner des objets. Ces objets pourront ensuite être envoyés dans Pokémon Rubis Oméga ou Pokémon Saphir Alpha.
Vous pouvez vous connecter sur le site Internet du Pokémon Global Link http://www.pokemon-gl.com/ et y enregistrer votre ID de Synchro-Jeu.
Veuillez noter que pour enregistrer votre ID, vous aurez besoin d'un compte sur le site Internet du Pokémon Global Link, exploité par The Pokémon Company et ses sociétés affiliées. Pour plus d'informations, veuillez consulter le site Internet du Pokémon Global Link.
Choisissez « Synchroniser » pour vous connecter au Pokémon Global Link. Attention : si vous n'avez pas enregistré votre ID de Synchro-Jeu sur le Pokémon Global Link, vous ne pourrez pas vous connecter et vous amuser sur le site.
Si vous activez la synchronisation automatique dans le menu « Changer les paramètres », celle-ci se fera sans intervention de votre part à chaque sauvegarde. Le PSS (système de recherche de joueurs) doit être connecté à Internet pour que cette option fonctionne.
Vous pouvez utiliser les Poké Miles que vous avez accumulés en les envoyant au Pokémon Global Link. Choisissez dans les paramètres l'option « Tout envoyer » ou « Ne pas envoyer ».
Lorsque vous recevez des objets obtenus sur le Pokémon Global Link via la Synchro-Jeu, une livreuse apparaît dans les Centres Pokémon. Parlez-lui pour obtenir vos objets.
Vous pouvez aussi envoyer les photos prises durant un Concours Pokémon Live ! Quand vous prenez une photo, souvenez-vous que vous pouvez utiliser la Synchro-Jeu !
Attention : vous ne pouvez sauvegarder sur la carte SD de votre console qu'une seule photo à envoyer via la Synchro-Jeu. Si vous prenez une autre photo avant d'avoir envoyé la précédente, cette dernière sera remplacée.
