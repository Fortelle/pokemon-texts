Le tableau détaille les altérations de statut dont peuvent souffrir les Pokémon au combat.
Laquelle voulez-vous consulter ?
Les Pokémon empoisonnés perdent régulièrement des PV tant qu'ils se battent. Le poison ne se dissipe pas, même une fois le combat terminé. L'Antidote guérit cette altération.
La paralysie réduit la Vitesse et peut empêcher d'utiliser les capacités. Elle dure même après le combat. L'Anti-Para permet de la guérir.
Les Pokémon endormis ne peuvent pas combattre. Il arrive qu'un Pokémon endormi se réveille de lui-même, mais dans le cas contraire, il continue à dormir même une fois le combat terminé. Utilisez un Réveil pour guérir cette altération.
Les brûlures réduisent non seulement l'Attaque des Pokémon, mais diminuent aussi peu à peu leurs PV. Cette altération perdure même après un combat. Un Anti-Brûle est le remède parfait.
Les Pokémon gelés ne peuvent plus rien faire. Il arrive qu'un Pokémon se dégèle de lui-même, mais dans le cas contraire, il reste gelé même une fois le combat terminé. L'Antigel réchauffera votre Pokémon !
