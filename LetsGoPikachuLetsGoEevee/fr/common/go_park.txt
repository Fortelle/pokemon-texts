Les GO Parks
La Zone de Jeu
Il est possible de transférer les Pokémon de Pokémon GO vers vos GO Parks. Pour ce faire, commencez par associer votre jeu à un compte Pokémon GO en ouvrant le menu PARAMÈTRES depuis le menu principal, puis sélectionnez CONNEXION AVEC POKÉMON GO. Choisissez ensuite les Pokémon à transférer depuis Pokémon GO. Attention : Une fois transférés, ces Pokémon ne peuvent pas être renvoyés vers Pokémon GO.
Les GO Parks sont des aires de jeu accueillant les Pokémon transférés depuis Pokémon GO. Vous pouvez attraper ces Pokémon ou les envoyer vers un autre parc en interagissant avec eux. Notez que les Pokémon que vous attrapez ne pourront ensuite pas retourner dans un GO Park. Pour sortir d'un parc, appuyez sur le bouton X.
Vous pourrez jouer dans la Zone de Jeu si vous réunissez 25 Pokémon de la même espèce dans vos GO Parks. Le but du jeu est simple : guider tous les Pokémon jusqu'au portail en haut de l'écran dans la limite du temps imparti. Attention : Certains Pokémon sortiront du sol pour vous mettre des bâtons dans les roues ! Si vous souhaitez interrompre une partie, appuyez sur le bouton X.
